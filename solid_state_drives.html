  SOLID STATE DRIVES (SSDS)


   What Is A Solid State Drive?

   An SSD is a computer component. Like the HDD, it is used by the computer to store
   files and data. Unlike the HDD, SSD’s have no moving parts, and instead use
   semiconductors and flash memory to store data. Flash drives use a collection of
   transistors stored in a collection of rows and columns to record data. If an electrical
   current passes through a chain of transistors, the value of the data is set to one. If
   the current is disrupted, the value is set to 0. Each chain of transistors can store up
   to one or two bits of data.

FUN FACT: A single MB today takes up 11 billion times less physical space than a MB did in the late 50's  


   History of the Solid State Drive

   The first SSD’s were produced in the 1950s, using different technologies to store data,    such as magnetic core technology and CCROS. Their usage quickly ceased though, with    the rise of cheaper storage devices. SSD’s saw some development and usage in the    1970 and 1980’s, but their high price made them unpopular. At that time, the SSD could    hold up to 2mb of data.
   By 1991, the storage was increased tenfold. In 1995, the SSD we know today that uses flash memory was introduced, which saw their
   use in military and aerospace applications. Since then, SSD’s have evolved into what we know today, with up to terabytes of storage
   capacity, and their sizes much smaller.

FUN FACT: One Gigabyte of data is over one billion bytes in total.
